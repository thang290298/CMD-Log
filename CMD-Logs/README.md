# CMD-Log
Tools ghi chép toàn bộ thao tác user SSH

Thực hiện cài đặt cấu hình
```sh 
curl -Lso- https://raw.githubusercontent.com/thang290298/CMD-Log/main/cmdlog.sh | bash
```
